# hackerClaire
Simple, fun program to discover bank accounts using Java with features of terminal input and GUI pop-ups. 
*This project is for learning only and doesn't access real bank info*

## Features
- Terminal-based user interaction using Scanner
- GUI prompt pop-up for Name and Password input
- Simulated bank alert
- Password decoding logic in separate class
- Timed delays for dramatic effect 

## Technologies
- Java

## Controls
- Keyboard input

## How to Run
- Compile Java files: 'javac TheBlackHatRises.java GUI.java Decode.java'
- Run program: 'java TheBlackHatRises'
